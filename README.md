# wbs-tool

HTMLだけで動くWBS（Work Breakdown Structure）/ ガントチャートツール。
`index.html` をブラウザで開くだけで使えます。サーバー・ビルド・インストール不要。

## 使い方

`index.html` をダブルクリックしてブラウザで開くだけ。

## 機能

- **階層タスク管理** — 大項目／中項目／小項目をインデントで管理（WBS番号 1.2.3 を自動採番）
- **ガントチャート** — 開始／終了日からバーを自動描画。土日シェーディング、今日の線、マイルストーン（開始＝終了）表示
- **進捗の自動集計** — 親タスクは子タスクから進捗・期間を自動ロールアップ
- **担当者・状態管理** — 未着手／進行中／完了／保留。進捗100%で自動的に完了に
- **自動保存** — ブラウザの localStorage に自動保存（次回開いたとき復元）
- **インポート／エクスポート** — JSON での保存・読込、CSV 出力（Excel対応・BOM付き）

## 操作

| 操作 | 方法 |
|------|------|
| タスク追加 | ヘッダーの「＋大項目」「子タスク」「同階層」ボタン、または各行の「＋」 |
| 階層変更 | 行選択 → 「→」「←」ボタン、または `Tab` / `Shift+Tab` |
| 並べ替え | 行選択 → 「↑」「↓」ボタン |
| 折りたたみ | タスク名左の「▼」をクリック |
| 削除 | 行の「✕」、またはヘッダーの「🗑」 |

データはすべてブラウザ内（localStorage）に保存され、外部送信は行いません。
